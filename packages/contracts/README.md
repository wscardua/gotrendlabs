# Contracts

Diretorio reservado para contratos compartilhaveis, como snapshots OpenAPI ou clientes gerados, somente quando houver uso real.

Nesta fase de fundacao, os contratos continuam como fonte de verdade em `docs/specs/contracts/` e pela documentacao viva da FastAPI em `/docs`.

Nao adicionar SDK ou cliente gerado sem uma feature que defina consumidor, processo de geracao e validacao em CI.
